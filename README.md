# Smart EV Visualisation

Frontend for the SmartEV project.

## Setup

Install dependencies:

```bash
npm install
```

Create a copy of the .env file

```bash
cp .env.example .env
```

Remember to update the MapTiler API key.
<br>
Go to https://cloud.maptiler.com/maps/
<br>
To generate an API key.

## Running the project

Command:

```bash
npm run dev
```
